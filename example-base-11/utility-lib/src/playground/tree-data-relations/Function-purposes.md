# getChildren

it has purpose to find solution for below use-case

```
suppose I have few objects in an array, having uniqueId, name and relations objects array. each relation object has fields like id, name, hasId, withId.

we have only one relation in the format of : object having uniqueId in 'hasId' field (say Object1) has relation with object having uniqueId in 'withId' field (say Object2), such that, below sentence can be formed.

"Object1 is 'father of' Object2".

Please write one javascript code to get all children for a given object's uniqueId

```
